# Palindrome Checker

This is a simple Python script to check if the entered text is a palindrome.
A Palindrome is a sentence that is same forwaard and backward.

## How to run

\`\`\`bash - 
python palindrome.py
\`\`\`

## Example

\`\`\`
Enter text: madam 
Palindrome!
\`\`\`
